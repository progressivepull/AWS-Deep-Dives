# 11.1 Learning objectives 

* Route 53 is AWS’s fully managed Domain Name System (DNS) service.

* It translates human‑readable domain names into the infrastructure addresses applications use.

* The service integrates tightly with other AWS offerings, allowing DNS answers to reflect the real‑time health of resources.

* It supports advanced routing policies such as latency‑based, geolocation, weighted, and failover routing.
