# 20.1 Learning objectives 
 
- **Lesson Overview**
  - Lesson 19 focuses on the **AWS Software Development Kits (SDKs)**

- **Problem the SDKs Solve**
  - Directly interacting with AWS APIs for every service is complex and error-prone
  - Managing authentication, request signing, and responses manually is inefficient

- **What AWS SDKs Provide**
  - Simplified interfaces for interacting with AWS services
  - Abstract away low-level API calls and authentication details
  - Enable rapid integration of AWS functionality into applications

- **Scope of the Lesson**
  - Overview of available **AWS SDKs**
  - How to use SDKs in application development
  - Where to find official **documentation and resources**

- **Learning Objective**
  - Understand how AWS SDKs streamline application development on AWS
