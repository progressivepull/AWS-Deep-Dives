# 24.1 Learning objectives 
 
- **Lesson Overview**
  - Lesson 23 focuses on **AWS Developer Tools**

- **Purpose of AWS Developer Tools**
  - Designed to support **developers** and **code management**
  - Enable automation across the software development lifecycle

- **Key Services Covered**
  - **CodeCommit**
    - Managed Git-based source control repository
  - **CodeBuild**
    - Builds and tests application code
  - **CodeDeploy**
    - Deploys built artifacts to target environments

- **CI/CD Focus**
  - Tools work together to support:
    - **Continuous Integration (CI)**
    - **Continuous Delivery / Deployment (CD)**

- **Lesson Format**
  - Each sub-lesson includes:
    - Conceptual slides
    - Hands-on demonstrations
  - Tools are demonstrated **together** to show real-world workflows

- **Learning Objective**
  - Understand how AWS Developer Tools integrate to manage, build, test, and deploy code
