# 18.1 Learning objectives 
 
 - **Lesson Overview**
  - Lesson 17 focuses on **AWS API Gateway**

- **What is AWS API Gateway**
  - A **fully managed service** for creating and managing APIs
  - Designed to be **secure**, **scalable**, and highly available

- **Core Functionality**
  - Acts as an **entry point** for applications
  - Routes requests to backend AWS services

- **Supported Backends**
  - **AWS Lambda**
  - **Amazon EC2**
  - **Amazon S3** (including file-based access)
  - Other AWS services

- **Key Capabilities**
  - API creation and management
  - Automatic scaling
  - Built-in security features

- **Topics Covered in the Lesson**
  - Different **API deployment options**
  - Using the **sample API** provided by AWS
  - **Automatic API Gateway creation** when integrating with AWS Lambda

- **Learning Objective**
  - Understand how API Gateway enables service integration and API-driven architectures
