# 21.1 Learning objectives 
 
- **Lesson Overview**
  - Lesson 20 focuses on **AWS CloudFormation**

- **What is AWS CloudFormation**
  - A service for **infrastructure as code (IaC)**
  - Uses a declarative language to define AWS resources

- **Core Capabilities**
  - Provision entire infrastructures with a single action
  - Supports resources such as:
    - Compute instances
    - Databases
    - Load balancers
    - Other AWS services

- **Key Benefits**
  - Consistent and repeatable infrastructure deployment
  - Simplifies resource management and automation

- **Topics Covered in the Lesson**
  - CloudFormation components
  - Creating infrastructure using CloudFormation
  - Updating existing infrastructure using CloudFormation

- **Learning Objective**
  - Understand how to provision and manage AWS infrastructure using CloudFormation
