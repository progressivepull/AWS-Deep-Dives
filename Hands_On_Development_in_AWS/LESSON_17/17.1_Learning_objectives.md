# 17.1 Learning objectives 
 
 - **Lesson Overview**
  - Lesson 16 focuses on **Amazon Kinesis**

- **What is Amazon Kinesis**
  - A **real-time data processing and analytics service**
  - Designed for ingesting and processing streaming data at scale

- **Types of Data Supported**
  - Real-time data such as:
    - Audio
    - Video
    - Other continuous data streams

- **Core Capabilities**
  - Enables **instant processing and analysis**
  - Helps derive insights from data as it is generated

- **Kinesis Sub-Services Covered**
  - **Kinesis Data Streams**
  - **Kinesis Data Firehose**
  - **Kinesis Data Analytics**

- **Learning Objective**
  - Understand the purpose and use cases of each Kinesis sub-service
