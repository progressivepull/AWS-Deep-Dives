# 19.1 Module 6 Developing in AWS introduction 
 
- **Module Overview**
  - Module 6 is titled **“Developing in AWS”**

- **Focus Shift**
  - Moves from simply using AWS services
  - To **developing applications that run on and interact with AWS**

- **Key Distinction**
  - Emphasizes application development rather than service configuration
  - Highlights how code integrates with AWS services

- **Core Topic**
  - Using the **AWS SDK** within applications
  - Enables applications to programmatically interact with AWS services

- **Learning Objective**
  - Understand how to write and run applications that leverage AWS capabilities
