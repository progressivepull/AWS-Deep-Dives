# 19.2 Learning objectives 
 
```markdown
- **Lesson Overview**
  - Lesson 18 focuses on the **AWS API**

- **What is the AWS API**
  - The foundational interface used to interact with AWS
  - All interactions with AWS services go through the AWS API

- **How the AWS API is Used**
  - Used by the **AWS Management Console**
  - Used by **AWS SDKs** and programmatic tools
  - Both rely on the same underlying API calls

- **Core Concept**
  - AWS API executes the **intent of user requests**
  - Ensures consistent behavior across all access methods

- **Lesson Objective**
  - Understand the AWS API
  - Learn different methods for interacting with AWS services
