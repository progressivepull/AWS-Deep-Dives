# 23.1 Learning objectives 
 
- **Lesson Overview**
  - Lesson 22 focuses on **Amazon Elastic Container Service (ECS)**

- **Why ECS Matters**
  - Containers and microservices are widely used in modern application architectures
  - ECS provides a managed way to run containerized workloads on AWS

- **What is Amazon ECS**
  - A service for the **creation, running, and management of containers**
  - Supports container-based and microservices architectures

- **Core Capabilities**
  - Orchestrates container deployments
  - Manages container lifecycles and scaling
  - Integrates with other AWS services

- **Lesson Objective**
  - Explore the fundamentals of **Elastic Container Service**
  - Understand how ECS supports containerized applications
